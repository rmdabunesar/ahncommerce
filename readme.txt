== Description ==

AhnCommerce is a WooCommerce-compatible e-commerce WordPress theme. It is an ideal choice for selling products online, featuring a stunning design and complete responsiveness across all devices. The theme includes support for custom logo, custom menus, featured images, footer widgets, and is translation-ready.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Install and activate WooCommerce to enable full e-commerce functionality.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

AhnCommerce includes support for WooCommerce for e-commerce functionality.

= How do I set up the home page? =

After installing the theme, create a new page and assign the "Home Page" template to it. Then go to Settings > Reading and set that page as your static front page.
